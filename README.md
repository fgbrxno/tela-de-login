## ✨ Features

* **Design Glassmorphism:** Interface moderna que utiliza o efeito de vidro fosco (`backdrop-filter`) para criar um visual elegante e translúcido sobre o plano de fundo.
* **Tipografia Customizada:** Integração com o Google Fonts utilizando a família tipográfica *Poppins*, garantindo um aspecto limpo e legibilidade.
* **Ícones Integrados:** Uso da biblioteca *Boxicons* para adicionar ícones visuais e intuitivos diretamente dentro dos campos de input.
* **Layout Fluido:** Estrutura construída com CSS Flexbox para manter o formulário perfeitamente centralizado na tela, independente da resolução.
* **Pronto para Integração:** Código HTML/CSS limpo e semântico, deixando o formulário de login perfeitamente estruturado para futura integração com banco de dados e sistemas de Back-End.
